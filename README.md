# DevTrack

DevTrack is a developer-focused project management system designed to organize projects, tasks, and engineering workflows.

The system demonstrates production-style full-stack engineering using the MERN stack.

---

## Features

- Project creation and management
- Task tracking within projects
- Task assignment to developers
- Task status workflow (todo → in-progress → done)
- JWT-based authentication

---

## Tech Stack

Backend
- Node.js
- Express
- MongoDB
- JWT Authentication

Frontend
- React
